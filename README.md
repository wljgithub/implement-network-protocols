## 目的

以前看书的时候觉得网络协议是一个很抽象的东西，似乎理我很遥远，特别是看到报文格式这些字眼的时候，好像这些东西只存在课本中。所以我希望用程序把所有研究过的协议实现出来，把它们变成能看得见的东西，我希望能真切地感受到它们的存在。

而且这些知识不是只存在课本上用于教学目的，而是实实在地影响着我们生活中方方面面的存在
